Zanata is a web-based translation platform for translators, content creators and developers to manage localisation projects.

Zanata is not currently being developed or maintained. While a community effort may be possible, it so far has not materialized.

I want to...

help translate

  • All you need is a web browser
  • Translate from any computer with an Internet connection
  • No command line
  • No source conflicts
  • Translation memory shows history of similar translations to reuse
  • See updates from other translators in seconds

translate content

  • Localise your documentation
  • Enable communities to translate your files
  • Accept translations without opening your source repository
  • Currently supported: HTML, subtitles, InDesign Markup Language, Publican DocBook, LibreOffice and plaintext. Read about currently supported raw documents

translate software

  • Localise your software interface and messages
  • Enable communities to translate your files
  • Accept translations without opening your source repository
  • Currently supported: GNU Gettext-based PO files, Java Properties, XLIFF, Qt Linguist, Mozilla DTD, JSON.
